# 🚀 Guía de Instalación y Despliegue
📋 Prerrequisitos
- Android Studio (versión Ladybug o superior)
- Java Development Kit (JDK) 17 o superior
- Un dispositivo Android físico con la depuración USB activada o un emulador configurado.
  
🛠️ Pasos para ejecutar el proyecto 
1. Clonar el repositorio: git clone https://github.com/BarreroDev/TuLista.git.
2. Abrir en Android Studio:Selecciona File ──► Open y busca la carpeta donde clonaste el proyecto.
3. Configuración de Firebase (Opcional): 
   - El proyecto ya incluye la configuración base, pero si deseas conectar tu propia base de datos, 
   debes descargar tu archivo google-services.json desde la consola de Firebase y colocarlo en la ruta app/.
4. Lanzar la aplicación:
- Conecta tu dispositivo móvil o inicia tu emulador.
- Haz clic en el botón Play verde (▶️) en la barra de herramientas superior para compilar e instalar la app.
